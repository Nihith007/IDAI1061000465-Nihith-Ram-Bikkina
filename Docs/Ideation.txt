SECTION 1 — IDEATION APPROACH
================================================================

Following the Empathize & Define stage, we conducted a structured
ideation session using the "How Might We" (HMW) framework derived
from our problem statement and empathy maps.

We generated 40+ ideas across 6 HMW categories, then prioritised
them using dot voting (impact vs. feasibility matrix).

================================================================
SECTION 2 — "HOW MIGHT WE" QUESTIONS
================================================================

HMW 1: How might we help employees decompress in under 15 minutes?
HMW 2: How might we remove the social stigma of needing a break?
HMW 3: How might we personalise the pod experience automatically?
HMW 4: How might we help employees build a consistent break habit?
HMW 5: How might we make the post-session feeling visible and trackable?
HMW 6: How might we accommodate different stress types and preferences?

================================================================

SECTION 4 — PRIORITISATION MATRIX
================================================================

Ideas evaluated on:
  IMPACT   — How significantly does it solve the core problem?
  EFFORT   — How feasible is it to prototype and implement?

HIGH IMPACT + LOW EFFORT (Do First — included in prototype):
  ✔ MoodSync AI recommendation engine
  ✔ Guided breathing exercises (Breathe with Me)
  ✔ Pod Scheduler with time slots
  ✔ Before/after stress journal
  ✔ Stress tracking dashboard with chart
  ✔ Preset library (4 named modes)
  ✔ Streak counter and badges

HIGH IMPACT + HIGH EFFORT (Consider for v2):
  → Calendar integration with Google Calendar
  → Real wearable device integration (Fitbit, Apple Watch)
  → Company-wide manager wellness dashboard
  → Fully physical pod hardware

LOW IMPACT + LOW EFFORT (Nice to have):
  → Seasonal scent rotation
  → Export data to PDF
  → Waitlist feature

LOW IMPACT + HIGH EFFORT (Deprioritised):
  → Kid-friendly mode
  → Anonymous leaderboard
  → Time-of-day aware presets (partially covered by AI)

SECTION 5 — 5 COMPULSORY FEATURES (Final Concept)
================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
FEATURE 1: MOODSYNC AI
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
What:    AI-driven pod configuration based on current stress level
How:     Combines mood survey (1-10), wearable HR/HRV data, and
         voice tone analysis to compute a stress score, then
         automatically selects music, lighting, aroma, and session
         duration from a personalised preference model
Why:     Research showed users don't know what they need in the
         moment and experience decision fatigue when stressed
Personas: Riya, Aarav, Neha, Rohan (high tech comfort, want AI)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
FEATURE 2: BREATHE WITH ME
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
What:    Guided micro-meditation with animated breathing circle
How:     User selects breathing technique (4-7-8 or box breathing),
         follows animated visual guide, completes 4 rounds in under
         3 minutes, then prompted to log their mood change
Why:     Only 20% of users actively meditate — a guided tool
         teaches the skill while providing immediate relief
Personas: Priya, Divya, Meera, Aarav (open to guided techniques)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
FEATURE 3: POD SCHEDULER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
What:    Advance booking system for pod time slots
How:     Displays available and booked time slots for each pod,
         allows user to select slot and duration (15/30/45 min),
         confirms booking and sends a reminder notification
Why:     70% cited time pressure as the main barrier — scheduling
         makes breaks feel legitimate and protected
Personas: Aarav, Karan, Sameer, Arjun (need structured routine)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
FEATURE 4: POST-SESSION JOURNAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
What:    Reflective post-session log with stress ratings and notes
How:     User rates stress before and after session (0-100 scale),
         gives a star rating, selects a mood tag, and writes a free-
         form reflection note — all stored privately on-device
Why:     Self-awareness is critical for sustainable stress management;
         journaling also closes the feedback loop for the AI model
Personas: Meera, Rohan, Priya, Divya (already journal at home)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
FEATURE 5: STRESS TRACKING DASHBOARD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
What:    Visual progress tracker showing stress trends over time
How:     Line chart shows stress scores before/after each session,
         stat cards show total sessions, minutes, streak, and avg
         stress reduction; badge system rewards consistent use;
         AI insight card generated from session history
Why:     Data-driven users need to see evidence that the pod works;
         visible improvement motivates continued use
Personas: Neha, Arjun, Sameer, Riya (data-motivated, need proof)

SECTION 6 — ADDITIONAL INNOVATIVE FEATURES (Beyond compulsory)
================================================================

FEATURE 6: Preset Library (4 named mood modes)
  - Sunset Calm: warm light + lo-fi + lavender
  - Forest Refresh: cool light + nature sounds + eucalyptus
  - Focus Flow: colour therapy + binaural beats + citrus
  - Deep Unwind: warm light + classical + sandalwood
  Why: Users have different stress types — presets remove choice
  fatigue while still offering personalisation.

FEATURE 7: AI Insight Card
  After 3+ sessions, the app analyses which preset + settings
  combination produced the best stress reduction for that user
  and displays a personalised insight on the home screen.

FEATURE 8: Streak & Badge System
  - First Session 🌱, 5-Day Streak 🔥, Zen Master 🧘, 10 Sessions ⭐
  Gamification encourages habitual use without being intrusive.

FEATURE 9: Counselor Nudge
  If a user records stress scores ≥ 8.0 for 3+ consecutive sessions,
  the app gently suggests speaking with an HR representative or
  wellness counselor — a safety net for severe burnout cases.
