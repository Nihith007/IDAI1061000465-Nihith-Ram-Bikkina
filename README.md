# Summative Assessment – Mindful Work: Smart Stress Relief System


## 📋 Project Scope

**Scenario:** Mindful Work is a mobile app connected to a Smart Stress Relief Pod that helps employees reduce workplace stress quickly. The app collects stress data and uses AI to personalize relaxation sessions using lighting, music, aroma, and meditation.

**Goal:** Provide fast, effective, and personalized stress relief in workplaces.

## 🧠 Design Thinking

**Empathize & Define**

**Causes of Stress:**

* Deadlines
* Heavy workload
* Long hours
* Burnout

**Personas:**

Aarav (Employee): Needs quick stress relief
Meera (Manager): Needs calm and focus

**Problem Statement:**
Employees need a quick and personalized way to reduce stress during work without losing productivity.

## 🚀 Core Features
* Mood check (emoji/slider)
* Heart rate via wearable
* Voice-based stress detection
* Smart pod control (lighting, music, aroma, meditation)
* AI-based personalized recommendations
## 📱 App Screens
* Home Screen → Start session
* Stress Check → Select mood
* Pod Setup → Customize environment
* Session Screen → Timer + breathing guide
* Progress Screen → Track improvement
## 🔄 User Flow
* User feels stressed
* Opens app
* Completes stress check
* AI suggests setup
* User enters pod
* Session runs
* User gives feedback
* AI improves recommendations
## 🤖 AI Personalization
* Learns from past sessions
* Uses feedback + stress data
* Improves suggestions over time
## 🎨 Design

* Colors: Soft blue, light green, white
* Font: Poppins + Roboto
* Style: Minimal, calming, easy to use

## 💡 Key Idea

AI + Smart Pod = Instant personalized stress relief in offices
